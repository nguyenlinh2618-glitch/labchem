⚡ Cấu hình Firebase

Cách lấy cấu hình:
1. Vào console.firebase.google.com → Tạo project mới
2. Thêm Web App → Sao chép firebaseConfig
3. Vào Realtime Database → Tạo database → Rules: ".read": true, ".write": true
4. Dán các giá trị vào đây
